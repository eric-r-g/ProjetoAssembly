REGISTRADORES UTILIZADOS NO PROGRAMA:

main:
$s0 = registrador com array principal
$s1 = tamanho da array
$t1 = índice temporário para scan_array
$v0 = usado para syscalls
$a0 = usado para endereços de strings

scan_array:
$t1 = índice atual do array
$t3 = valor numérico lido do usuário
$t4 = offset de memória calculado (índice * 4)
$t5 = endereço calculado do elemento no array
$s0 = endereço base do array (herdado de main)
$s1 = tamanho do array (herdado de main)

merge_sort:
(Não implementado ainda - registradores a definir)

merge:
$s0 = endereço do array principal
$s1 = endereço do l_array (subarray esquerdo)
$s2 = endereço do r_array (subarray direito)
$s3 = tamanho do l_array
$s4 = tamanho do r_array

loop_principal:
$t0 = índice do array principal
$t1 = índice do l_array (subarray esquerdo)
$t2 = índice do r_array (subarray direito)
$t3 = índice ajustado do array principal
$t4 = índice ajustado/valor do l_array
$t5 = índice ajustado/valor do r_array

merge_final:
$t0 = índice do array principal (continuação)
$t1 = índice do l_array (para loop_esq)
$t2 = índice do r_array (para loop_dir)
$t3 = índice ajustado do array principal
$t4/t5 = índices ajustados/valores dos subarrays

loop_esq:
$t0 = índice do array principal
$t1 = índice do l_array
$t3 = índice ajustado do array principal
$t4 = índice ajustado/valor do l_array

loop_dir:
$t0 = índice do array principal
$t2 = índice do r_array
$t3 = índice ajustado do array principal
$t5 = índice ajustado/valor do r_array

imprimir:
$t1 = índice inicial da array
$v0 = usado para syscalls
$a0 = usado para valores/strings

print_array:
$s0 = endereço base do array (herdado de main)
$s1 = tamanho da array (herdado de main)
$t1 = índice atual da array
$t4 = offset de memória calculado (índice * 4)
$t5 = endereço calculado do elemento no array
$a0 = usado para valores a imprimir

end:
$v0 = usado para syscall de término
